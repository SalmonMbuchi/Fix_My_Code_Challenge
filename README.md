# Fix My Code

Fix my code is a new type of project where we jump into an existing code base and fix it. Sometimes you will know the language, sometimes you will not. The goal is not to recode everything, just debug and fix it.

## Tasks

0. **FizzBuzz**
Debug the implementation of FizzBuzz in python.

1. **Print square**
Debug the implementation of printing a square in Javascript.

2. **Sort**
Debug the implementation of sorting arguments in Ruby.

3. **User password**
Debug the implementation of a User class in Python.

4. **Double linked list**
Debug the implemetaion of a Double linked list in C.
